Low Res Image

While developing scenes, we noticed that intersecting spheres looked like the balloon portion of hot air balloons so we were inspired to create a scene that incorporated these. Initially, we were thinking about using a teapot as a basket because of the scene in Alice in Wonderland that has something similar. Due to the difficulty of this, we decided to just use ply meshes of hot air balloons. Our scene was constructed by loading in some meshes with transformations attached to them and then giving these our materials.

    How to run

    Inside the raytracer.cpp file set the tracer that you want on line 24. Then Inside world.cpp include the proper build file that you would like to run. All thats left is to run ./make.sh inside the raytracer directory which will compile and then run tracer. We have included a progression timer so you can see how close your image is to rendering. You can then open the ppm as you see fit.
